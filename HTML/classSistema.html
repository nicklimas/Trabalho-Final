◆ login()

Pessoa * Sistema::login ( )

Metodo para logar a pessoa no sistema.

Le da entrada um email e uma senha. Compara a entrada com as informações no arquivo de login. Caso as informações batam, o login é efetuado com sucesso. Caso não, a função termina. Retorna um ponteiro do tipo pessoa com as informações do usuario logado. Em caso de erro ao abrir os arquivos ou usuario nao encontrado, joga uma exceção.

Returns
Pessoa* ponteiro do tipo pessoa com as informações do usuario logado.
